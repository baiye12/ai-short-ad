# Interaction Patterns 完整模式库

> 本文件为 ai-short-ad 技能的互动模式参考。SKILL.md 中只保留常用速查，本文件收录所有模式、适用场景、组合玩法和写作要点。

## 角色反转类（适合儿童产品）

核心逻辑：孩子扮演成人角色，家长处于被动位，反差越大越好笑。

| Pattern | Description | Example |
| --- | --- | --- |
| Role reversal | Kid acts as authority figure, parent is helpless | Kid as "fitness coach" scolding dad's form |
| Bait-and-switch | Setup suggests concern, real motive is selfish | Kid "diagnoses" dad, actually wants the milk |
| Form mismatch | Formal/serious framing for trivial content | Business negotiation over a snack trade |
| First-move advantage | Act before getting permission | Drink the milk first, then ask about the chips |
| Fake expertise | Kid uses jargon confidently but absurdly | "Your developmental window has closed, Old Li" |
| 假装关心实为抢食 | 表面担心家长身体，实际想自己吃 | "爸你这个血压不能再吃咸了，这包薯片我帮你解决" |
| 小大人训话 | 用成人训斥语气指责家长幼稚行为 | "都多大的人了还跟孩子抢零食" |
| 假装成熟做决策 | 替家长做生活决策，语气不容置疑 | "从今天起家里的饮食归我管" |

### 写作要点

- 孩子的台词要用成人语境词，但内容保持孩子视角
- 家长的反应要真实（无奈、困惑、哑口无言），不要配合表演
- 反转点设置在产品口播位前2-3秒，让产品成为笑点收尾

## 成人对话类（适合全品类）

核心逻辑：双方态度/认知/立场形成强烈反差，越一本正经越有喜剧效果。

| Pattern | Description | Example |
| --- | --- | --- |
| 情绪反差 | 一方急一方淡，反差越大越好笑 | 家长急追问，营养师头都不抬淡定回答 |
| 权威碾压 | 专业人士一句话堵死外行，越平静越有压迫感 | 营养师："可以啊，到时候跟不上别打他就行" |
| 冷幽默面瘫 | 用最平淡的语气说最炸的话 | 客服面无表情："这产品确实不适合您，建议您继续受着" |
| 误读升级 | 双方理解不在一个频道，越解释越离谱 | 家长问添加剂，营养师以为是问加价 |
| 反向推销 | 销售/客服拼命劝退，越劝退越想买 | "一天一条就够了，多了浪费"→ 家长反而更信 |
| 一本正经胡说 | 用极其专业的语气包装荒谬逻辑 | "根据我们的研究，这个产品的分子结构适合..." |
| 误解升级螺旋 | 双方都在认真对话，但理解完全错位，越聊越离谱 | 顾客问保质期，客服以为是问保寿命 |
| 冷静破坏 | 极度冷静的角色用一句话戳破对方情绪 | 急脾气的客户vs面瘫客服，客服一句话让对方哑口无言 |

### 写作要点

- "淡"的一方台词越短越好，"急"的一方台词可以长
- 反差要在前两句内建立，否则笑点不出来
- 产品口播位放在"淡"方的最后一句话，用专业语气收尾

## 场景错位类（适合所有品类）

核心逻辑：把产品放到完全不匹配的场景，制造荒谬感。

| Pattern | Description | Example |
| --- | --- | --- |
| 古装现代穿帮 | 古代场景用现代产品，角色当理所当然 | 大臣上朝掏出一罐儿童奶粉 |
| 严肃场合闹剧 | 庄重场合出现完全不搭的产品 | 法庭审理现场，法官掏出一包零食 |
| 倒置时空 | 古人在做现代事，或现代人在做古代事 | 穿西装的CEO用文言文讨论零食分配 |
| 仪式感错配 | 用极其庄重的仪式对待日常小事 | 给一包薯片举行加冕典礼 |

### 写作要点

- 仪式感越足，反差越大，笑点越炸
- 角色对荒谬完全无感，一本正经地推进
- 产品口播位用"仪式主持者"的台词引出

## 节奏控制

### 单模式 vs 组合模式

- 15秒广告：单模式足够，不要贪多
- 18-20秒广告：可叠加两种模式，但必须有主次
- 拆两条的广告：每条独立用一种模式

### 反转点位置

| 模式 | 反转点位置 | 原因 |
| --- | --- | --- |
| 角色反转类 | Shot 3-4 | 前面铺人物关系，反转后接产品收尾 |
| 成人对话类 | Shot 2-3 | 反差要早建立，后面靠台词节奏推笑点 |
| 场景错位类 | Shot 1开场即建立 | 荒谬感要第一时间抓住注意力 |

### 产品口播位衔接

- 反转高潮后2秒内进入产品口播位
- 口播位的台词要和前面的笑点逻辑呼应，不要硬切
- 旁白tagline保持和笑点同一语气

## 模式选择快速判断

| 产品类型 | 推荐模式 | 原因 |
| --- | --- | --- |
| 儿童奶粉/营养品 | 角色反转类 | 孩子演大人天然有反差 |

| 保健品 | 成人对话类 | 营养师/医生角色天然适配 |
