# 完整品类合规清单

> 本文件为 ai-short-ad 技能的完整审核合规参考。SKILL.md 中只保留核心速查表，本文件收录所有品类的详细规则、反面案例和正面范本。

## 使用方法

1. 在 Write scripts 步骤之前，根据产品品类定位对应专项
2. 写完台词后，逐条对照本清单正向核对：每句台词是否落在"正确表述"范畴内
3. 如有歧义，落在范畴外即视为风险，改写后再核

## 通用规则（适用所有品类）

| 类目 | 错误表述 | 正确表述 |
| --- | --- | --- |
| 功效宣称 | 直接宣称治愈/治疗/解决某问题 | 陪伴、全面营养、好口感等生活方式表述 |
| 效果对比 | 使用前后对比、量化承诺（数字/时间） | 生活场景片段，呈现自然使用状态 |
| 医疗用语 | 处方、治疗、临床、诊断等（台词中作为玩笑可以） | 日常用语，如"每天一杯""营养补充" |
| 绝对保证 | "一定能""保证""100%"等绝对化表述 | "陪伴每一步""营养支持" |
| 品牌滥用 | 用品牌名暗示必然效果 | 品牌名仅作品牌标识出现 |
| 绝对化定性 | "解决XX""根治XX""告别XX" | 改善、助力、搭配等柔性动词 |

## 品类专项规则

### 儿童奶粉/营养品

| 错误表述 | 正确表述 |
| --- | --- |
| 直接说"长高""增高" | 陪伴成长、全面营养 |
| 身高前后对比、量身高画面 | 日常生活片段 |
| "解决挑食""治挑食" | 好喝、孩子爱喝 |
| "聪明""提升智力" | 营养支持成长阶段 |

**正面范本**：
- "每天一杯MN儿童奶粉，陪伴成长的每个早晨"
- "孩子爱喝的奶香口感，营养一步到位"

### 零食/食品

聚焦：口感、原料、食用场景

| 错误表述 | 正确表述 |
| --- | --- |
| 宣称减肥/美容/治疗功效 | 突出原料产地、工艺口感 |
| 暗示替代正餐 | 场景化呈现（下午茶、加班补给） |
| "0卡0糖"绝对化承诺 | 贴检测报告原文，原样引用数据 |

### 日用品/家电

聚焦：功能贴检测报告原文

| 错误表述 | 正确表述 |
| --- | --- |
| 宣称杀菌/治病 | 引用检测报告具体数值 |
| "彻底去除""100%除菌" | "经XX检测，除菌率达XX%"（附报告编号） |
| 无数据支撑的性能承诺 | 描述使用场景和体验感受 |

### 美妆/护肤

聚焦：肤感、使用体验、妆容效果

| 错误表述 | 正确表述 |
| --- | --- |
| 宣称医疗美容效果 | 描述上脸肤感、质地、延展性 |
| "速效""根治""医美级" | "持续使用XX天后的使用感受" |
| 前后对比量化承诺 | 呈现自然妆容状态、使用过程 |

### 保健品/膳食补充剂

聚焦：营养成分、食用便利性、生活方式匹配

| 错误表述 | 正确表述 |
| --- | --- |
| "治疗""治愈"某疾病 | "补充XX营养素" |
| 替代药物暗示 | "日常营养补充，按需食用" |
| 量化疗效承诺 | 标注成分含量，贴检测数据 |

### 3C数码/家电智能

聚焦：使用体验、场景匹配、智能联动

| 错误表述 | 正确表述 |
| --- | --- |
| "最好""最强""颠覆"等绝对化 | 具体功能、参数、使用场景描述 |
| 虚假对比竞品 | 呈现独家功能点 |
| "永久免费""终身"等承诺 | 具体服务期限和范围 |

## 正向核对流程（交付前）

1. 列出所有台词、旁白、字幕
2. 逐句标定所属范畴：生活方式 / 陪伴 / 口感 / 营养支持 / 使用体验 / 场景描述
3. 如有句子无法归入任何范畴 → 风险，改写
4. 涉及数据的句子 → 核对是否贴检测报告原文
5. 涉及品牌名的句子 → 核对是否仅作品牌标识，未暗示必然效果
6. 全部通过 → 交付；任一未通过 → 回到Write步骤改写后再核
