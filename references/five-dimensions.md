# 五维度注意力权重分配：完整理论

> 本文件为 ai-short-ad 技能的提示词科学参考。SKILL.md 中只保留广告场景下的速查版，本文件收录完整理论、坐标系详解、非广告场景映射和案例分析。

## 核心原理

AI视频从随机噪声逐步去噪生成，提示词 = 刻刀方向。

- 描述越多 → 刀越乱 → 特征污染 → 画面失控
- 结构化锚点 = 精准赋值 → 去噪路径聚焦 → 画面可控

**顶级提示词不拼字数，拼谁能最精准地分配模型的交叉注意力权重。**

## 五维度控制坐标系

| 维度 | 含义 | 参考图能否锁死 |
|------|------|----------------|
| 1. 绝对主体与物理动势 | 画面中的人物/物体及运动状态 | 有主体参考图时锁死 |
| 2. 环境场与情绪光影 | 场景、光调、氛围 | 有场景参考图时锁死 |
| 3. 光学与摄影机调度 | 镜头运动方式 | **参考图永远无法提供** |
| 4. 时间轴与状态演变 | 变身/生长/爆炸等时间变化 | 静态图 = 锁死为静止 |
| 5. 美学介质与底层渲染参数 | 画风、色调、渲染风格 | 有风格参考图时锁死 |

## 实操心法：5秒5维度扫描法

拿到任何画面需求，5秒内用5维度扫描，识别哪些参数已被素材锁死、哪些还是空白，**只对空白维度精准下刀**。

### 做减法场景（信息熵拉满的底图）

极致克制，只写摄影机调度指令，一个多余形容词都不加。

锁死的维度再写 = 文本特征和图片特征抢权重 → 特征污染 → 原画细节被破坏。

### 做加法场景（需变身/动作/状态演变）

按维度展开，每帧物理状态都写清楚，用结构化锚点（面部/服装/场景/时间分段）精准分配注意力。

## 前置条件：素材清晰度

模糊的原图会被AI视觉编码器误判为特殊艺术风格，并在后续每帧死锁这个错误特征。

**必须先用 Topaz Video/Photo 提升素材清晰度后再喂给模型。**

## 广告场景下的维度映射

广告生产中：

- 首帧图/场景参考图已锁死 **维度1**（主体外貌+动势）和 **维度2**（环境+光影）
- **维度4** 通常为静止（单场景15s）
- **维度5** 由参考图风格决定

**提示词只需要补全**：
- 维度3（镜头调度）
- 维度4中需要的动作变化

这就是为什么广告提示词只写景别+动作+台词，不重复描述外貌。

## 非广告场景的维度映射

### 变身/特效场景

- 维度1：主体身份不变，但状态变化（人类→狼人）
- 维度4：变身过程需逐帧写清
- 提示词重心：维度4 时间轴状态演变

### 风格迁移场景

- 维度5由参考图决定，其他维度按内容补全
- 提示词重心：维度1+维度3

### 多镜头叙事场景

- 每个Shot独立做5维度扫描
- 维度3在每个Shot都要明确（参考图不跨Shot）

## 案例分析

### 案例1：饮料广告（做减法）

**素材**：首帧图已含人物、产品、场景
**锁死维度**：1、2、5
**空白维度**：3（镜头）、4（动作）

**提示词**：
```
中景 | 角色拿起饮料罐，递向镜头 | 台词一
特写 | 罐口贴近镜头，气泡上升 | 台词二
```

只写维度3（景别）+ 维度4（动作），其他不加。

### 案例2：变身效果（做加法）

**素材**：人物首帧图
**锁死维度**：1（身份）、5（画风）
**空白维度**：2（场景）、3（镜头）、4（变身过程）

**提示词**：
```
场景：暗夜森林，月光从树叶间漏下
中景 | 角色站立，面部开始出现纹路
特写 | 纹路蔓延至全身，瞳孔变为金色
远景 | 完成变身，角色仰天长啸
```

维度2（场景）+ 维度3（景别）+ 维度4（变身过程）都写清楚。

## 反模式

### 反模式1：信息过载

```
ERROR: 一个美丽的中国古代宫廷场景，金色的阳光洒下，
女主角穿着红色绣金凤袍，头戴金钗，手持玉杯...
```

问题：维度1、2、5全在文字描述，和参考图抢权重 → 特征污染。

### 反模式2：情绪形容词过多

```
ERROR: 温馨感人的画面，温暖的光线，幸福的表情...
```

问题："温馨""感人""温暖""幸福"都是情绪描述，模型无法映射到具体视觉特征。

### 反模式3：跨 Shot 因果链

```
ERROR: Shot 01 角色因为Shot 03的事情感到悲伤...
```

问题：模型按 Shot 独立生成，跨 Shot 因果链无法传递。
